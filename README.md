# 厕所信息卡片项目

## 项目简介
这是一个用于展示和管理厕所信息的Web应用程序。用户可以查看厕所的位置、设施情况以及提交相关点评。

## 功能特点
- 展示厕所基本信息
- 支持无障碍设施标识
- 用户点评系统
- 数据持久化存储

## 技术栈
- 前端：
- 图标：Font Awesome
- 后端：Node.js
- 数据库：MongoDB

## 开发指南

### 环境要求
- Node.js
- MongoDB

### 安装依赖
```bash
npm install
```

### 开发服务器
```bash
npm run dev
```

### 构建项目
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```
